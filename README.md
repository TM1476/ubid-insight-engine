# UBID Insight Engine 🚀

### The Problem
Government data is siloed across Labour, Tax, and Utility departments, leading to ghost entities and revenue leakage[cite: 1].

### Our Solution
A **Stateless Entity Resolution Engine** that merges department data in real-time without storing sensitive PII on the server[cite: 1].

### Key Features
- **Deterministic Linkage:** Merges records based on fuzzy name matching[cite: 1].
- **Dormancy Pulse:** Uses activity signals to identify "Shell Companies"[cite: 1].
- **Mobile-First UI:** Optimized for field inspectors in the industrial belt[cite: 1].

### Tech Stack
- **Backend:** FastAPI (Python)
- **Frontend:** React + Tailwind CSS
- **Deployment:** Render (Serverless/No-Storage)
