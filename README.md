# UBID Insight Engine 🚀

### The Problem
Government data is siloed across Labour, Tax, and Utility departments, leading to ghost entities and revenue leakage.

### Our Solution
A **Stateless Entity Resolution Engine** that merges department data in real-time without storing sensitive PII on the server.

### Key Features
- **Deterministic Linkage:** Merges records based on fuzzy name matching.
- **Dormancy Pulse:** Uses activity signals to identify "Shell Companies".
- **Mobile-First UI:** Optimized for field inspectors in the industrial belt.

### Tech Stack
- **Backend:** FastAPI (Python)
- **Frontend:** React + Tailwind CSS
- **Deployment:** Render (Serverless/No-Storage)
